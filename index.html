1. Compter le nombre de chambres total.

SELECT COUNT (chb_numero) FROM t_chambre;

2. Retrouver le prix maximal d’une chambre.

SELECT MAX (trf_chb_prix) FROM tj_trf_chb;

3. Retrouver le nombre de places totales dans l'hôtel.

SELECT SUM (chb_couchage) FROM t_chambre;

4. Retrouver le prix moyen des chambres.

SELECT AVG (trf_chb_prix) FROM tj_trf_chb;

5. Compter le nombre de clients dont l’enseigne est renseignée.

SELECT COUNT (cli_nom) FROM t_client WHERE cli_enseigne IS NOT NULL;

6. Retrouver le prix moyen des chambres durant la période du 15/01/2000 au 15/06/2000.

7. Compter le nombre de clients par genre. (Nombre de M., de Mme. et de Melle. en une seule requête).

SELECT tit_code,COUNT (tit_code) FROM t_client GROUP BY tit_code;

8. Compter le nombre de chambres par étage.

SELECT COUNT(chb_etage),chb_etage FROM t_chambre GROUP BY chb_etage;

9. Retrouver le nombre de places totales dans l'hôtel par étage.

SELECT SUM (chb_couchage),chb_etage FROM t_chambre GROUP BY chb_etage;

Retrouver le nombre de places totales dans l'hôtel par étage seulement si ce nombre est supérieur à 15

SELECT SUM (chb_couchage),chb_etage FROM t_chambre GROUP BY chb_etage HAVING SUM (chb_couchage)>15;
